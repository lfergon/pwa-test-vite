# Arch Counter App

## Instructions

```bash
pnpm run install
```

### Run the app

```bash
pnpm run dev
```

### How to turn your existing Vue 3 App into a PWA
```bash
vue add pwa
```

Install: [https://vite-plugin-pwa.netlify.app/guide/](https://vite-plugin-pwa.netlify.app/guide/)
